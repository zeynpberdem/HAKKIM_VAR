# 🤖 Hakkım Var - Akıllı Ajanlar (Agents)

Bu klasör, platformun arka planında çalışan otomasyon ve analiz birimlerini içerir.

### 🔍 Mevcut Ajanlar
1. **Legal Parser Agent (`legal_parser.js`):** - Kullanıcının girdiği doğal dil metinlerini analiz eder.
   - Metin içerisinden hukuki anahtar kelimeleri (tazminat, ihtarname, mobbing vb.) ayıklar.
   - Sorunu ilgili hukuk kategorisine (İş Hukuku, Tüketici Hukuku vb.) yönlendirir.
2. **Draft Generator Agent:** - Analiz edilen verilere göre otomatik dilekçe taslağı hazırlar.
